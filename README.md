# Kubernetes-Examples

This repository contains **enterprise-ready Kubernetes examples**, organized by API type (workloads, services, ingress, security, etc.). Each directory focuses on one concept with production-friendly YAML.

## Directories

- workloads/  # Deployments, StatefulSets, DaemonSets, Jobs, CronJobs
- services/   # ClusterIP, NodePort, LoadBalancer, Headless, ExternalName

Future additions:
- ingress/
- config/
- security/
- storage/
- autoscaling/
